# Blockchain Real Estate Market

by James Kwon Lee
An Ethereum-based DApp that enables users to buy and sell property and verify ownership without revealing sensitive details.

Real Estate Marketplace on the Blockchain

by James Kwon Lee

First uploaded: December 7, 2019.

In a developed country, you might take property ownership, legal paperwork, and chains of title for granted. But what if you live in a nation riddled with corruption and instability? What if the government cannot even prove the validity of your contracts and ownership of property in court? What if there is no reliable justice system in which attorneys can fight and prove your ownership via contracts?

My decentralized App aims to mitigate the problems mentioned above by leveraging the power of code-based smart contracts, on the Ethereum blockchain. I minted unique ERC-721 tokens that represent homes and property and uploaded them to the marketplace. I use Zokrates and Zero-Knowledge proofs to prove the existence and ownership of property without revealing its contents.

The properties are listed on Opensea.io.

Versions Used: Solidity Version: ^0.5.1; Node v10.16.3 Truffle v5.1.1 (core: 5.1.1)

To get started, please:
```
npm install
```

To run tests, please CD to eth-contracts and then run the following:
```
truffle test
or
npm test
```
